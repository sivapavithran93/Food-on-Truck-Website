# Food On Truck

A simple food ordering website created using HTML and CSS. This project displays popular fast-food items, allows users to view menu options, and includes a contact form for customer inquiries.

## Features

* Clean and simple navigation bar
* Food menu with images and descriptions
* Order Now buttons for each item
* Contact form with basic validation
* Contact details section
* Hover effects for better user interaction

## Technologies Used

* HTML5
* CSS3

## About the Project

This project was built to practice HTML page structure, CSS styling, form creation, and hover effects. The goal was to create a simple restaurant-style landing page with an attractive menu layout and contact section.
